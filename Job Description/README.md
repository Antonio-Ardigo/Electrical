# Job Descriptions — WSM Electrical SPOF Assessment

Role profiles for the WSM engineering subcontract supporting the **Electrical Single
Point of Failure (SPOF) Assessment** of **21 assessment units** across the Central &
Eastern sector (Riyadh, Madinah, Qassim, Eastern Region) groundwater & surface-water
desalination plants, over a **4-month** contract.

Aligned to the WSM Subcontractor Technical Specification
**`WSM-SUBC-SPEC-001 Rev 0`**
([WSM Subcontractor Technical Specification Rev 0.docx](WSM%20Subcontractor%20Technical%20Specification%20Rev%200.docx)).

| Profile | Per spec | File |
|---------|----------|------|
| **Project Manager (Electrical)** | §3.1 — 1 off · BSc EE (MSc/PMP adv.) · ≥10 yrs | [project-manager-electrical.md](project-manager-electrical.md) |
| **Electrical Engineer** (all calculations) | §3.1 — 2 off · BSc EE · ≥5 yrs | [electrical-engineer.md](electrical-engineer.md) |
| **Electrical Technician** (site survey / data capture) | *not in §3.1 headcount — optional field-support role* | [electrical-technician.md](electrical-technician.md) |

**Common requirements** (per spec §3.2 / §6): LV & MV experience; plant assessment
(SPOF / reliability / condition); **SCE accreditation** (to be discussed); **Arabic &
English**; QHSE (ISO 9001/14001/45001) and host permit-to-work; non-intrusive surveys;
NDA and **NCA cybersecurity (v6.0)** data handling.

**Study method:** SF-01 Document Inventory · SF-02 FMECA · SF-03 Recommendations ·
SF-04 Site Visit Report.
**Calculation set:** load-flow · short-circuit · protection coordination · arc-flash
(IEEE 1584) · earthing & lightning-protection.
**Standards:** IEC, IEEE (incl. 1584 / 493), NFPA 70E, ANSI, NEMA; IEC 60812 (FMECA);
ISO 20815 / ISO 14224.

> The spec headcount is **1 PM + 2 Electrical Engineers**; the technician profile is
> provided per request as an optional site-survey support role and is not part of the
> fixed personnel supply.
