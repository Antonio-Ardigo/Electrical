About This Manual

This field manual is a self-paced reference for a junior electrical engineer assigned to assess the main electrical equipment of a medium-voltage / low-voltage (MV/LV) industrial process plant with an indicative demand of approximately 2 MW. It covers equipment identification, core electrical calculations, single-point-of-failure (SPOF) analysis, and practical recommendations.

How to Use This Manual

Work through the parts in order. Each part builds on the previous: start by identifying the equipment and understanding its function (Part 1), apply the key calculations (Part 2), then conduct the SPOF analysis (Part 3) and formulate recommendations (Part 4).

Use the sidebar to navigate between sections. Checklists in each part track your progress — tick items as you complete them; your progress is saved automatically in the browser. Print the manual (use the Print button) for field use; the sidebar is hidden and each section starts on a new page.

This manual is a single offline HTML file. It requires no internet connection and no installed software beyond a modern web browser. Checklist state is stored in the browser's local storage — use the same browser session to preserve progress.

Scope and Plant Configuration

The indicative plant configuration used throughout this manual is:

Supply13.8 kV SEC incomer, 60 Hz, metered at MV level
MV switchgear13.8 kV switchgear (17.5 kV class) with numerical protection relays
Transformers2 × 1600 kVA, Dyn11, 13.8 kV / 0.4 kV, Z = 6 % (TX-1 and TX-2)
LV main switchboardSplit-bus, 400/230 V TN-S, with normally-open bus-tie (LV-MSB)
Motor controlMultiple MCCs fed from LV-MSB sections; 4-pole motors at 1800 rpm
Standby generationDiesel generator 1250 kVA ESP (DG) with automatic transfer switch (ATS)
EarthingMV resistance-earthed via NER (400 A, 19.9 Ω); LV TN-S
Environment50 °C design ambient, dust-laden desert site (Saudi Arabia)
Total demand~2 MW (indicative); apparent power ~2.35 MVA at pf 0.85

Refer to the master single-line diagram (SLD) throughout for equipment locations and tag references.

What's New in Rev 3

  • Saudi (SEC) localization — electrical basis changed to 13.8 kV MV, 400/230 V LV (TN-S), 60 Hz, 50 °C design ambient; standards cited as SASO-adopted IEC plus SEC Distribution Code, Saudi Grid Code and SBC 401.
  • Part 2 rewritten as deep-dive tutorials — every calculation topic now follows an eight-step teaching pattern (why it exists → failure example → feel for the numbers → worked calculation → settings → technology → control → sizing), with new topics: per-unit, breakers, arc flash, lightning/SPD, UPS and ATS.
  • NEW Part 6 — Equipment Catalogues — commercial vendor families (switchgear, transformers, breakers, MCC/VFD, UPS/DC, gensets/ATS, relays and auxiliaries) with "how to read the datasheet" guides and selections for this plant. All catalogue data is indicative; no part numbers.
  • New orientation sections — the Saudi electrical context (SEC, SASO, SABER, SBC 401) and designing for 50 °C and dust.
  • Concepts companion — a separate self-contained file (concepts-rev3.html) explaining the core concepts behind each calculation.
