About This Manual

This field manual is a self-paced reference for a junior electrical engineer assigned to assess the main electrical equipment of a medium-voltage / low-voltage (MV/LV) industrial process plant with an indicative demand of approximately 2 MW. It covers equipment identification, core electrical calculations, single-point-of-failure (SPOF) analysis, and practical recommendations.

How to Use This Manual

Work through the parts in order. Each part builds on the previous: start by identifying the equipment and understanding its function (Part 1), apply the key calculations (Part 2), then conduct the SPOF analysis (Part 3) and formulate recommendations (Part 4).

Use the sidebar to navigate between sections. Checklists in each part track your progress — tick items as you complete them; your progress is saved automatically in the browser. Print the manual (use the Print button) for field use; the sidebar is hidden and each section starts on a new page.

This manual is a single offline HTML file. It requires no internet connection and no installed software beyond a modern web browser. Checklist state is stored in the browser's local storage — use the same browser session to preserve progress.

Scope and Plant Configuration

The indicative plant configuration used throughout this manual is:

Supply11 kV utility incomer, metered at MV level
MV switchgear11 kV ring-main unit (RMU) with protection relay
Transformers2 × 1600 kVA, Dyn11, 11 kV / 415 V (TX-1 and TX-2)
LV main switchboardSplit-bus, 415 V, with normally-open bus-tie (LV-MSB)
Motor controlMultiple MCCs fed from LV-MSB sections
Standby generationDiesel generator (DG) with automatic transfer switch (ATS)
Total demand~2 MW (indicative); apparent power ~2.35 MVA at pf 0.85
